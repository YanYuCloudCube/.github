### ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🔒 **隐私优先** | 所有数据本地存储，无第三方追踪 |
| 🚀 **零配置启动** | 克隆即可使用，无需数据库 |
| 📴 **离线可用** | PWA 支持，断网可用 |
| 🎨 **高度可定制** | 主题、Logo、模型自由配置 |
| 🤖 **AI 本地化** | 支持 Ollama 本地模型 |

### 🛠️ 技术栈

- **六大 LLM Provider**：Ollama / OpenAI / 智谱 GLM / 通义千问 / DeepSeek
- **18+ 功能面板**：拖拽系统，支持合并/拆分/浮动
- **完整测试**：2,702 个测试用例，100% 通过率
