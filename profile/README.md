## 🚀 旗舰项目：YYC³-Family-AI

**开源本地优先的 AI 智能编程助手 | 数据主权，完全归你**

<div align="center">
  <img src="https://raw.githubusercontent.com/YanYuCloudCube/YYC3-Family-AI/main/public/Family-AI-001.png" alt="YYC³ Family AI 界面预览" width="800" style="border-radius: 16px; box-shadow: 0 20px 35px -10px rgba(0,0,0,0.3);" />
</div>

<br/>

> **🔥 核心理念：数据主权在你手中** —— YYC³-Family-AI 是一个完全运行在浏览器中的本地开发工具，所有数据存储在你的设备上，无需注册账号，无需云端依赖。

### ✨ 核心特性一览

| 特性 | 说明 | 状态 |
|------|------|------|
| 🔒 **隐私优先** | 所有数据本地存储 (IndexedDB + localStorage)，无第三方追踪 | ✅ |
| 🚀 **零配置启动** | 克隆仓库即可使用，无需数据库、无需后端服务 | ✅ |
| 📴 **离线可用** | PWA 支持，断网环境下仍可正常工作 | ✅ |
| 🎨 **完全可定制** | Logo、标语、主题、模型、面板自由配置 | ✅ |
| 🆓 **完全免费** | MIT 开源协议，可自由使用、修改、分发 | ✅ |
| 🤖 **AI 本地化** | 支持 Ollama 本地模型，数据不出本机 | ✅ |

### 🛠️ 技术亮点

| 技术模块 | 说明 |
|---------|------|
| **六大 LLM Provider** | Ollama / OpenAI / 智谱 GLM / 通义千问 / DeepSeek / 自定义 |
| **18+ 功能面板** | 基于 react-dnd 的拖拽系统，支持合并/拆分/浮动 |
| **三栏 IDE 布局** | AI 对话 + 文件管理 + 代码编辑，可自由调整 |
| **AI 代码生成流水线** | Context → SystemPrompt → LLM SSE → CodeApplicator → Diff Preview |
| **完整测试覆盖** | 104 个测试文件，2,702 个测试用例，100% 通过率 |
| **存储架构** | 三层混合存储（IndexedDB + localStorage + Zustand） |

## 🌟 快速开始

### 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/YanYuCloudCube/YYC3-Family-AI.git
cd YYC3-Family-AI

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 访问 http://localhost:5173
```

### 🎨 在线体验

> **🌐 立即体验：[https://family-ai.yyccube.com](https://family-ai.yyccube.com)**

无需安装，打开浏览器即可使用！

### 📦 生产构建

```bash
pnpm build      # 构建生产版本
pnpm preview    # 本地预览构建结果
```
